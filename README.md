# mcfalloutBot-shop_item

## 簡介
  
這是一個在臺灣Minecraft伺服器-廢土伺服器中可使用的Bot

該Bot具備了幾項功能與指令：
- 隨傳隨到: /tpahere, /tpa
- 開始兌換shop_item的項目: /m BotID start 兌換ID
- 停止兌換: /m BotID stop
- 將Bot身上的物品全數丟出: /m BotID throw
